reverse-engineered xal token generator for epic games talon sdk.

xal is the fingerprint token used during epic login. 15 fields get collected, shuffled (mt19937), hashed (fnv-1a), compressed (zlib), encrypted (chacha20), then base64'd.

## token format

```
base64(nonce[12] + random[4 LE] + chacha20_ciphertext)
```

shuffle order and chacha nonce are derived from the random u32 embedded in each token.

## usage

1. grab `xal.js` from talon sdk, drop it in root
2. `node extractor/deob.js` — deobfuscates + extracts chacha20 key/nonce into `config.json`
3. `node gen.js` — spits out a ready-to-use xal token

key rotates daily so grab a fresh xal.js each time.

### debug tools

```
node debug/decoder.js <base64_xal_token>
node debug/encoder.js [config.json] [xal_data.json]
```

## files

| file | desc |
|---|---|
| `config.json` | chacha20 key + nonce (auto-generated by extractor) |

| `extractor/deob.js` | deobfuscator + key extractor |
| `debug/decoder.js` | decode xal token back to json |
| `debug/encoder.js` | encode json into xal token |
